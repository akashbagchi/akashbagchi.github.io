Animal Welfare, Care & Aid Mobile Application

The Animal Welface, Care & Aid Mobile Application is an app for Android and iOS developed in Python using KivyMD. The primary objective of this application is to consolidate the various services and features animal welfare activists and enthusiasts require, but are unable to find and make use of on existing platforms. The application is a one-stop hub for animal adoptions, vaccinations, helpline numbers, donating to welfare drives etc. View on GitHub

Web Technologies Project

University Student Resources Portal

The University Student Resources Portal, designed in this case for use in Dayananda Sagar University - is a fully responsive and modern web application built with HTML, Bootstrap, JavaScript, MySQL and PHP. The website provides attending students of the university with a wide host of services and resources. Class notes, important announcements, semester timetables, Club events and social media links, etc. It also provides admin & maintenance personnel to upload the aforementioned resources on-site with the use of a password-protected form mechanism. UI/UX Introductory Project

"Blu-Tuna" Guitar Tuner Application

The "Blu-Tuna" mobile application is a design concept for an application that can be used to tune guitars, ukuleles and other stringed instruments. It features a playful and communicative design with lively colors and a user-friendly interface. The app provides tuner and metronome functionalities - two of the basic yet powerful tools in any budding musician's arsenal. This design project was undertaken as part of The California Institute of the Arts - "Elements of User Interface Design" course.

Technologies used:

  • Adobe XD
  • Gooogle Fonts
